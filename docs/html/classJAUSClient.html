Detailed Description

Abstract transport between the FORT agent and a JAUS-capable vehicle.

Concrete implementations expose discovery, control negotiation, heartbeat monitoring, and wrench-effort streaming. The bridge and state machine rely solely on this interface so it can be substituted in tests.
